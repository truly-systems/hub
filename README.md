Instalação
----------

Realizar o git clone do projeto:

`git@github.com:truly-systems/hub.git`

Entrar na URL onde foi

Alterar o  arquivo `config.php` as seguintes linhas:

``````````php
$dados_api = array(
		"host" => "http://endereco_glpi_aqui",
		"app_token" => "seu_app_token_aqui"
	);
``````````
